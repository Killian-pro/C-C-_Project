Bonjour,
C'est un Jeu du style Space invader! 
je me suis aidé du tp "particule" pour reprendre les vecteurs.
Les controles sont affichés dans la page settings du Menu (F2).
Pour jouer il suffit d'appuyé sur F1.
Vous avez seulement 5 vies et vous les perdrez 1 à 1 si les vaisseaux touche la Terre!
Le sort de cette planete depend de vous , tachez de ne pas toucher leur boucliers ils vous feront 
perdres des points !
Rapportez un maximum de points en allant le plus loin dans le Jeu.
